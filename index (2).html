What you are seeing

Each cell is colored by the predicted share from the logistic model, recomputed after solving the fixed point for μ.

How scroll controls the model

As each narrative step becomes active, the page sends slider values to the interactive frame using postMessage.

Inputs and standardization

The model uses the same training means and standard deviations for z-scoring, so scenarios stay comparable.

Notes and limitations

Treat this as a scenario explorer, not a forecast. Consider sensitivity checks for coefficients and data coverage.
